\hypertarget{gtsummary}{%
\subsection{GTSummary}\label{gtsummary}}

Gtsummary is a relatively new table package with great capabilities that
works with pipe syntax and various other table packages (flextable,
kableExtra, knitr, huxtable).

\hypertarget{syntax}{%
\subsection{Syntax}\label{syntax}}

Several base functions that create default tables with minimal input

\begin{itemize}
\tightlist
\item
  tbl\_summary()

  \begin{itemize}
  \tightlist
  \item
    Great for quick, fast table 1's that can be edited to your hearts
    content
  \end{itemize}
\item
  tbl\_regression()

  \begin{itemize}
  \item
    Creates quick, fast table to summarize regression models
  \item
    Also supports survival models, and Bayesian models from the brms
    package
  \end{itemize}
\item
  tbl\_uvregression()

  \begin{itemize}
  \tightlist
  \item
    Creates quick, fast (noticing a theme here?) tables for univariate
    analysis
  \end{itemize}
\end{itemize}
